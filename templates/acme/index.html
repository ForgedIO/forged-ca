ACME

Automated Certificate Management Environment — how your servers get leaf certs from this Issuing CA without manual CSR shuffling.

Slice 2 lights this up.
  • Start the step-ca daemon and serve the ACME directory over HTTPS
  • Default forgedca-acme provisioner bound to a "Web Server (Server + optional Client Auth)" template
  • Configurable leaf lifetime — defaults to short (30–90 days, industry-typical) with renew-at-2/3-lifetime auto-timer
  • Client Onboarding page with the ACME directory URL + copy-paste commands for step, certbot, acme.sh, cert-manager, Traefik, Caddy, nginx
  • HTTP-01 validation always; DNS-01 as a plugin (Cloudflare + Route53 first)
  • Admin UI's own cert transitions to ACME auto-renewal on the Settings page once this slice lands
