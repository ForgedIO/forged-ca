{# OS-specific Root-install instructions. Included by wizard/finish.html and core/home.html so the same guidance appears anywhere the admin has to trust the Root before clicking a "Sign Web UI" button. #}
Windows — system-wide (covers Chrome + Edge)
  1. Double-click the downloaded root_ca.crt file.
  2. Click Install Certificate.
  3. Choose Local Machine (requires admin) or Current User, then Next.
  4. Pick Place all certificates in the following store → Browse → Trusted Root Certification Authorities.
  5. Finish. Chrome, Edge, and most Windows apps now trust your CA.
macOS — system-wide (covers Safari + Chrome + Edge)
  1. Double-click root_ca.crt. Keychain Access opens.
  2. Select the System keychain when prompted.
  3. Find the new "ForgedCA Root CA" entry and double-click it.
  4. Expand Trust and set When using this certificate to Always Trust.
  5. Close the window (enter your password to save).
Linux — Debian / Ubuntu 
sudo cp root_ca.crt /usr/local/share/ca-certificates/forgedca-root.crt
sudo update-ca-certificates
Linux — RHEL / Rocky / CentOS Stream 
sudo cp root_ca.crt /etc/pki/ca-trust/source/anchors/forgedca-root.crt
sudo update-ca-trust
Firefox — maintains its own trust store (do this in addition to the OS step above)
  1. Open about:preferences#privacy (or Settings → Privacy & Security).
  2. Scroll to Certificates and click View Certificates….
  3. On the Authorities tab, click Import… and pick root_ca.crt.
  4. Tick Trust this CA to identify websites, click OK.
  5. Close the certificate manager. Firefox now trusts your CA.
