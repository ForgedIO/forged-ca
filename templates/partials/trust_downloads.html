Note on formats. Every file below is PEM-encoded text — base64-wrapped between -----BEGIN CERTIFICATE----- / -----END CERTIFICATE-----. A .crt and a .pem with the same content are interchangeable; Windows tooling tends to expect .crt, Unix tooling tends to expect .pem. You can open any of them in a text editor.

Automate with curl / Ansible / CI

The trust endpoints are public by default so unattended tooling can pull them. You can flip the node-level toggle to require admin auth if your environment demands it.

# Fetch the chain
curl -ko chain.pem https://{{ config.hostname|default:"localhost" }}:8443{% url 'trust:chain_pem' %}

# Fetch just the Root for fleet trust
curl -ko root_ca.crt https://{{ config.hostname|default:"localhost" }}:8443{% url 'trust:root_crt' %}
